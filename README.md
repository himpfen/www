# Himpfen Jekyll Site

Production-ready Jekyll site for the Himpfen umbrella brand.

## Includes

- Jekyll structure
- Bootstrap-based UI
- Approved gradient hero design
- Homepage
- Company pages
- Selected Work page
- Principles page
- Blog index
- Four production-ready blog posts
- SEO, feed, and sitemap plugin configuration

## Run locally

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000`.
